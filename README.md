Project Overview

This project demonstrates common SQL operations for a library management system, including CRUD operations, data retrieval, and analysis.

Skills Demonstrated

- Create: Insert new book records into the books table.

- Read: Retrieve information about members, issued books, and book categories.

- Update: Modify existing member addresses.

- Delete: Remove issued records.

- Aggregations: Count books issued per employee, total rental income by category.

- Joins: Identify books not yet returned.

Tasks Covered

- Add a new book record to the books table.

- Update an existing member address.

- Delete a record from issued_status.

- Retrieve all books issued by a specific employee.

- List members who have issued more than one book.

- Retrieve all books in a specific category.

- Calculate total rental income by book category.

- List books that have not been returned (using LEFT JOIN).

Dataset / Tables

- books: Contains book information (ISBN, title, category, rental price, status, author, publisher).

- members: Contains member information.

- issued_status: Tracks which books have been issued.
  
- return_status: Tracks returned books.

return_status: Tracks returned books.
